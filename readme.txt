== Description ==

GeoTheme To GeoDirectory plugin provides tool to convert geotheme directory listings in to geodirectory directory listings.
It converts listings, categories, tags, images, reviews, ratings, prices, claim listings data etc.

== Installation ==

You can install this plugin directly from your WordPress dashboard:

1. Go to the *Plugins* menu and click *Add New*.
2. Search for *GeoTheme To GeoDirectory*.
3. Click *Install Now* next to the GeoTheme To GeoDirectory plugin.
4. Activate the plugin.

Alternatively, see the guide to [Manually Installing Plugins](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
